8.23面经：复习JVM

JVM作用复习

JVM是java虚拟机，是java实现跨平台特性的原因，位于java和操作系统之间的虚拟计算机，有自己的堆，栈，PC寄存器等

java通过将java源文件编译成.class文件，再根据不同的平台使用JVM解释，转换成对应的机器码实现跨平台操作

因此java被称为半解释语言

JVM的构成

包含 类装载器子系统，运行时数据区，执行引擎

类加载子系统负责将.class文件加载到JVM中生成Class实例，包含应用程序类，第三方包，jdbc驱动包等

如果链表中存在环 ，则返回 true 。 否则，返回 false 。
